API Documentation

Integrate background removal into your applications with our simple REST API.

Endpoint Method Description
/api-keys POST Generate a new API key
/remove-background POST Remove background from image
/health GET Check API health status

Example Usage

curl -X POST "https://web-production-faaf.up.railway.app/remove-background" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -F "file=@image.jpg" \
  -o "result.png"
